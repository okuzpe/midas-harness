# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1, as its Code of Conduct.

In short: be respectful, welcoming, and constructive. Treat others with empathy.
Harassment and exclusionary behavior are not tolerated.

## Reporting
Report unacceptable behavior privately via the repository's
[Security Advisories](https://github.com/okuzpe/midas-harness/security/advisories/new)
or a direct message to the maintainers. Reports are handled confidentially and in
good faith.

The full text, scope, and enforcement guidelines are at the link above.
